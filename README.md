# אפליקציית בדיקות גנים

קובץ יחיד (`index.html`) – עובד בדפדפן, ללא שרת.

## שימוש מהטלפון (אחרי העלאה ל-GitHub)

1. העלו את התיקייה ל-GitHub (ריפו חדש).
2. בהגדרות הריפו: **Settings** → **Pages** → ב-**Source** בחרו **Deploy from a branch**.
3. ב-**Branch** בחרו `main` (או `master`) ותיקייה **/ (root)** → **Save**.
4. אחרי דקה־שתיים האפליקציה תהיה זמינה בכתובת:
   ```
   https://<שם-המשתמש>.github.io/ganei-tikva/
   ```
5. **בטלפון:** פתחו את הכתובת בדפדפן (Chrome/Safari) → תפריט (⋮) → **הוסף למסך הבית**.
6. מהאייקון במסך הבית האפליקציה תעבוד מלא, כולל שמירת דוחות (localStorage).
